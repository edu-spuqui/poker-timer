This is a timer of poker blinds which plays a sound when blinds change.
The config button on the top right corner allows for blind structure configuration.
How to input the blind structure:
One level per line, in the following format: integer1:integer2 or string:integer2 where:
  1. integer1 is the big blind value for that level;
  2. string is the denomination to be displayed for break or custom levels;
  3. (optional) :integer2 is the custom duration for that level in minutes.
